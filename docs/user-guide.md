# User Guide

这份文档面向只想安装和使用码住的人，不要求会 Android Studio、Supabase 或命令行。

## 安装 App

1. 打开项目的 GitHub Releases 页面。
2. 下载最新版本的 `mazhu-*.apk`。
3. 把 APK 安装到 Android 手机。
4. 如果系统提示“未知来源应用”，按系统提示允许本次安装。

码住目前不在应用商店发布，因此需要通过 APK 安装。

## 收藏公众号文章

### 方式一：分享给码住

1. 在微信中打开公众号文章。
2. 点击右上角分享或更多打开方式。
3. 选择“码住”。
4. 选择一个或多个收藏夹。
5. 点击完成。

### 方式二：复制链接后打开码住

1. 在微信文章中复制链接。
2. 打开码住。
3. App 识别到微信公众号链接后会询问是否添加收藏。
4. 选择收藏夹并保存。

如果文章已经收藏过，码住会提示已收藏，并允许继续加入其他收藏夹。

## 收藏夹和最近收藏

首页有两个主要视图：

- 收藏夹：按分类查看文章。
- 最近收藏：按时间查看最近保存的文章。

文章可以同时属于多个收藏夹。收藏夹内删除和最近收藏内删除的含义不同：

- 从收藏夹中移除：只移除当前收藏夹里的归属关系。
- 删除该文章：从所有收藏夹中删除这篇文章。

## 云同步

码住可以不登录使用。未登录时，收藏数据只保存在手机本地。

登录后：

- 收藏夹和文章会同步到云端。
- 换手机或重装后可以通过账号恢复云端数据。
- 桌面端 CLI 和 Skill 可以读取云端数据，用于 AI 问答。

网络不可用时，文章会先保存在本地，并显示待同步状态。网络恢复后可以手动同步。

## AI 问答能力

AI 问答不是手机 App 内置功能，而是桌面端增强能力。

你需要：

1. 在手机 App 中登录并同步收藏。
2. 在电脑上配置 `mazhu` CLI。
3. 安装 `mazhu-knowledge` Skill。
4. 在 Codex 或兼容 agent 中让它查询你的码住收藏夹。

开发者指南中包含 CLI 和 Skill 的安装方式。
